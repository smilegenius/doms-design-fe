# DOMS Design FE

Frontend design prototype for the SmileGenius Dental Operations Suite.

## Live Preview Setup

Follow these steps to run the live preview locally using the Claude Code Desktop app:

1. **Clone this repo** to your local machine:

   ```bash
   git clone https://github.com/smilegenius/doms-design-fe
   ```

2. **Open the Claude Code Desktop app.**

3. Go to the **Code** tab and **start a new session**.

4. **Select the repo folder** (`doms-design-fe`) as the working directory.

5. Type **`live preview`** and hit **Enter**.

6. Click the **Open browser** icon in the top-right corner to view the running app.
