# Lucide Icons — Used in This Project

Total unique icons: **131**

Every icon below is imported from the `lucide-react` npm package — there are no separate SVG files for them in the repo, they render as inline SVGs from the package.

## How to use them

```bash
npm install lucide-react
```

```tsx
import { Mail, AlertCircle, ChevronDown } from 'lucide-react';

<Mail className="w-4 h-4 text-blue-600" />
```

## How to grab the raw SVG files

For each icon below the URL format is `https://lucide.dev/icons/<slug>` — the page has a Download SVG button. The plain SVG files are also in the lucide repo at `https://github.com/lucide-icons/lucide/tree/main/icons`.



## Raw SVG files

All icons listed below are also included as plain `.svg` files under `lucide-svg/`. Each filename matches the slug column (kebab-case).
